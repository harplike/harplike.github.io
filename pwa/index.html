Background Change

Want to learn about PWA... keep reading!

You will create a simple page (named index.html) that will have a heading, and at least two paragraphs where you will be explaining what are the main characteristics of a PWA (Progressive Web Application). You will also give to the user a drop-down select so the user can select the type of background should be applied to the page - light blue or light gold - once the user clicks the button to change the background or as soon as the user selects light blue or light gold, you will use the respective image as background image of the page (the images you found in hw4 folder).
